FortressV2 is a Secure AI Boundary and Assurance Platform.

Terraform provisions a private cloud boundary using VNet/VPC isolation,
private endpoints, managed identity, and controlled egress.

The Fortress control plane continuously verifies that:
- Data and secrets cannot leave the boundary
- No public access paths exist
- Identity and secrets remain protected

The system produces audit-ready evidence while ensuring that
no PII or secret values are ever exposed.

Primary Users:
- Cloud / AI Platform Engineers
- Security / DevSecOps
- Risk & Audit teams
